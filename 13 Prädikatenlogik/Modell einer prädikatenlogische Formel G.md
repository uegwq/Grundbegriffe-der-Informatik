13 Modell einer prädikatenlogische Formel G
---
Ist (D, I) eine Interpretation für eine Prädikatenlogsiche Formel G, dann ist (D, I) ein Modell von G, wenn die Formel für jede Variablenbelegung auf Wahr auswertet.
